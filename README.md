# Backend Node.js + Express

Este proyecto es una API básica desarrollada con Node.js y Express que implementa separación de responsabilidades (MVC) y middlewares.

## Instrucciones de ejecución

1. Clonar el repositorio.
2. Abrir la terminal en la carpeta del proyecto.
3. Instalar las dependencias ejecutando: `npm install`
4. Iniciar el servidor ejecutando: `npm start` o `node app.js`
5. El servidor correrá en `http://localhost:3000`

## Endpoints disponibles

* **GET `/api/status`**: Retorna el estado del servidor.
* **POST `/api/contacto`**: Recibe un JSON con `nombre` y `mensaje`.
